# Agent Skills

A personal collection of skills for AI coding agents. Skills are packaged instructions and references that extend agent capabilities.

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### drizzle-orm-v1

Type-safe SQL ORM for TypeScript (v1 RC) with zero runtime overhead. Covers schema definition, relational queries v2, migrations, and validation.

**Use when:**
- Working with drizzle-orm v1 or `drizzle-orm@beta`
- Defining schemas, relations, or relational queries
- Migrating from Prisma to Drizzle
- Optimizing database performance in serverless/edge environments

## Installation

```bash
npx skills add mattdrose/skills
```
